## 🚀 Getting Started

### Prerequisites
- Node.js v18+
- Groq API Key (free at [console.groq.com](https://console.groq.com))

### Installation
```bash
git clone https://github.com/ibprahim-dev/cv-analyzer.git
cd cv-analyzer
npm install
```

### Setup
Create a `.env` file:
```
GROQ_API_KEY=your_api_key_here
```

### Run
```bash
npm start
```

Open `http://localhost:3000` 🎉
